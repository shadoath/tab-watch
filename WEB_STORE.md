### Full Description
(Plain text, no markdown)

```
TabWatch shows you exactly how long each browser tab has been open — down to the minute. If you keep tabs open as reminders or placeholders, this makes the problem impossible to ignore.

CORE FEATURES

Time tracking — every tab is timestamped the moment it loads. Durations update live while the popup is open and persist across browser restarts. Refreshing a page keeps the timer running; navigating to a new URL resets it.

Visit counter — tracks how many times you have activated each tab. Persists even after the tab is closed and reopened, so you can see which tabs you keep coming back to versus ones you have not touched in days.

Sort and search — sort by duration (longest open first, by default) or by title. Click the column headers to toggle direction. Search filters by title or URL instantly.

Jump or close — click any row to switch to that tab and window. Hover to reveal an X button that closes the tab without switching to it.

Tab groups — if you use Chrome tab groups, TabWatch shows each tab's group name and color as a pill.

Dark and light mode — toggle in the toolbar or the settings page. Preference is saved.

Keyboard navigation — use arrow keys to move through the list, Enter to jump to a tab, Escape to close the popup, and keys 1 through 5 to jump directly to the first five tabs.

SETTINGS

Open the settings page to configure:

- Badge count: show the number of open tabs on the extension icon
- Tab groups: show or hide group pills
- Tab limit warning: dim tabs that have been open longer than a set number of days (default: 7)
- Auto-refresh interval: 1, 3, 5, or 10 seconds
- Animations: fade-in effect when the popup opens
- Clear tab history: remove all stored timestamps and visit data

PRIVACY

TabWatch stores everything locally in your browser using Chrome's built-in storage. No data is ever collected, transmitted, or shared. No account required.
```
